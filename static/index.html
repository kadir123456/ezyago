Sıkça Sorulan Sorular

Merak ettiğiniz soruların cevapları burada

Bot nasıl çalışır?

Botumuz tescilli "Ezyago Akıllı Algoritması" ile piyasa verilerini analiz eder ve belirlediğiniz zaman diliminde trend değişimlerini tespit ettiğinde otomatik olarak pozisyon açar.

API anahtarlarım güvenli mi?

Evet, API anahtarlarınız güçlü şifreleme ile korunur. Sadece işlem yapma yetkisi istenir, para çekme yetkisi asla talep edilmez.

Minimum yatırım tutarı nedir?

Bot varsayılan olarak 25 USDT ile 10x kaldıraç kullanır, ancak bu ayarları kendi stratejinize göre değiştirebilirsiniz. Binance hesabınızda en az 100-200 USDT bulundurmanızı öneriyoruz.

Zarar durumunda ne olur?

Her işlemde belirlediğiniz stop-loss oranında otomatik koruma kurulur. Ayrıca take-profit hedefine ulaşıldığında pozisyon otomatik olarak kapatılır.

İptal etmek istersem ne yapmalıyım?

İstediğiniz zaman hesabınızı kapatabilir veya botu durdurabilirsiniz. Taahhüt yoktur, otomatik yenileme yoktur.

Hangi borsayı destekliyorsunuz?

Şu an için sadece Binance Vadeli İşlemler (Binance Futures) platformunu destekliyoruz. Binance hesabınız yoksa buradan oluşturabilirsiniz.

Bot ayarlarımı değiştirebilir miyim?

Evet! Kullanıcı panelinizdeki "Bot Kontrolü" sayfasından işlem miktarı, kaldıraç, stop-loss ve take-profit gibi ayarları kendi risk stratejinize göre düzenleyebilirsiniz.

API Anahtarı Nasıl Alınır?

  1. Binance hesabınıza giriş yapın
  2. Profil > API Management'a gidin
  3. "Create API" butonuna tıklayın
  4. Sadece "Enable Futures" seçeneğini işaretleyin
  5. API Key ve Secret'ı kopyalayıp buraya yapıştırın
