Sıkça Sorulan Sorular

Merak ettiğiniz soruların cevapları burada

Bot nasıl çalışır?

Botumuz EMA (9,21) teknik analiz stratejisini kullanarak 15 dakikalık periyotlarda piyasayı analiz eder. Trend değişimlerini tespit ettiğinde otomatik olarak long veya short pozisyon açar.

API anahtarlarım güvenli mi?

Evet, API anahtarlarınız güçlü şifreleme ile korunur. Sadece işlem yapma yetkisi istenir, para çekme yetkisi asla talep edilmez.

Minimum yatırım tutarı nedir?

Bot her işlemde 25 USDT ile 10x kaldıraç kullanır. Bu nedenle Binance hesabınızda en az 100-200 USDT bulundurmanızı öneriyoruz.

Zarar durumunda ne olur?

Her işlemde %4 stop-loss otomatik olarak kurulur. Bu sayede maksimum zarar sınırlanır ve hesabınız korunur.

İptal etmek istersem ne yapmalıyım?

İstediğiniz zaman hesabınızı kapatabilir veya botu durdurabilirsiniz. Taahhüt yoktur, otomatik yenileme yoktur.

API Anahtarı Nasıl Alınır?

  1. Binance hesabınıza giriş yapın
  2. Profil > API Management'a gidin
  3. "Create API" butonuna tıklayın
  4. Sadece "Enable Futures" seçeneğini işaretleyin
  5. API Key ve Secret'ı kopyalayıp buraya yapıştırın
